## 📖 Descripción

Sitio web desarrollado para la campaña política de **Alonso Henao Giraldo**, candidato al Concejo Municipal para el periodo 2023-2027.

La página presenta información sobre el candidato, experiencia laboral, preparación académica, propuestas, instrucciones de votación y canales de contacto, proporcionando una plataforma digital para acercar la campaña a la comunidad.

## 🚀 Características

- Diseño web responsivo.
- Menú de navegación interactivo.
- Información personal y profesional del candidato.
- Sección explicativa sobre cómo votar.
- Resumen de experiencia laboral y formación.
- Redes sociales integradas.
- Animaciones con JavaScript.
- Diseño moderno con HTML, CSS y JavaScript.

## 🛠️ Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts

## 🎯 Objetivo del Proyecto

Brindar una herramienta digital para divulgar la campaña política de Alonso Henao Giraldo, facilitando el acceso a la información del candidato y promoviendo la participación ciudadana.

## ✨ Funcionalidades

- Navegación por secciones.
- Menú adaptable para dispositivos móviles.
- Integración con Facebook y WhatsApp.
- Presentación de experiencia y perfil profesional.
- Información detallada del proceso de votación.
- Animaciones dinámicas con JavaScript.
